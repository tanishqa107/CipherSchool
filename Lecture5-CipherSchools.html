CSS Selectors CSS provides an artistic way of enhancing web pages CSS selector basically selects a html element or a set of html element thhat we want to style we can divide the css selectors in five types- simple, combinator, pseudo-class, pseudo-element, attribute element selector is a kind of simple selector which selects the element with a certain tag name for linking our css file to the html file we can write p{ text align : center; } for aligning the text to the center to sleect a particular paragraph to align give a particular id to that

and then in css file style it as #mypara { text align: center; } for using a class selector we use a full stop character or a period character followed by the class name .center-paragraph { text-align: center; } rememeber to use the same class name as center-paragraph in the html file paragraph as

p { background-color: cyan; } in this case all the paragraphs under the div tag will be coloures cyan, but if we want it to be seperated we can use the section tag instead of the div tag for adjacent sibling selector we use the + sign, it is used to select the element which is directly next to the other specific eleement they must have the same parent element and adjacent mean immeditaely following div + p { background-collor: red; } in this suppose we end the div tag somewhere like and immediately on the next line we write a p tag so the p tag will be coloured only as it right next to the div tag general sibling selector uses the negate symbol ~ it basically selects all the elementsthat are the next siblings of the specified element div ~ p { background-color: blue } now comes pseudoclass selector, a pseudo class is used to define a special state of the element, whenever we want to style a special state of the element we use this this works as in suppose we want to style something when mouse is over that certain element then we use this div ~ p: :hover { background-color: green; } now coming to pseudo element selector, it is used to style specified parts of an element div ~ p::first-letter { color: blueviolet; } now coming to the attribute selector, it is possible to style html elements that have specific attribute or attribute values a[target] { background-color: aqua; } for every anchor tag which contains the target attribute, the background colour will be aqua p{ text-align: center ; } #mypara { text-align: center; } .center-paragraph { text-align: center; } # { font-family: 'Times New Roman', Times, serif; } h1, .center-paragraph, p { text-align: center; color: aqua; } div p { background-color: brown; } div > p { background-color: aqua; } div + p { background-color: red; } div ~ p { background-color: blue } div ~ p: :hover { background-color: green; } div ~ p::first-letter { color: blueviolet; } a[target] { background-color: aqua; }
